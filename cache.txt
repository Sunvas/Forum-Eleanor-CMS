Звездочка * ниже означает имя форума по конфигу

Перечень всех кэшей:
1. *_moders_mmIDLANGUAGE - (mm - MultiModerator) Кэш модераторов при просмотре топиков внутри форума - для мультимодерации
2. *_moders_taIDLANGUAGE - (ta - Topic Access) Кэш модераторов с правами доступа к теме: при подписке на неактивные темы, при пометке неактивной темы прочтенной
3. *_moders_pIDLANGUAGE - (p - Post) Кэш модераторов при посте (создание / правка темы / поста)
4. *_moders_tIDLANGUAGE - (t - Topic) Кэш модераторов при просмотре темы
5. *_moders_fpIDLANGUAGE - (fp - Find Post) Кэш модераторов при поиске поста

Куки форума:
*-fr:TIMESTAMP - даты прочтения форумов
*-tr:ID-TIMESTAMP,ID-TIMESTAMP - даты прочтения тем
*-ar:TIMESTAMP - дата прочтенности всего форума
*-gt (Guest topics) : ID,ID,ID - перечень ID тем, которые принадлежат гостю
*-gts (Guest topics sign) : string - подпись предыдущего пункта от подделки
*-gp (Guest posts) : ID,ID,ID - перечень ID постов, которые принадлежат гостю
*-gps (Guest posts sign) : string - подпись предыдущего пункта от подделки.
